# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/filipebteixeira98/dt-money.git
```

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```

Go to http://localhost:3000/ to see the result.
